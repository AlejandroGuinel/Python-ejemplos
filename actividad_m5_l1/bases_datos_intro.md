# Bases de Datos - Introducción

## 1. El rol de una base de datos

Una base de datos relacional cumple un rol fundamental dentro de una empresa u organización, ya que permite **almacenar, organizar y gestionar información de manera estructurada y segura**. Gracias a esto, los datos pueden ser consultados, actualizados y analizados de forma eficiente, facilitando la toma de decisiones.

Además, ayuda a evitar la duplicación de información y asegura la integridad de los datos.

### Ejemplos de uso:

* **Sistema de ventas:** registro de clientes, productos y transacciones.
* **Gestión de usuarios:** almacenamiento de cuentas, contraseñas y perfiles.
* **Inventario:** control de stock, productos disponibles y movimientos.

---

## 2. Características de un RDBMS

Un **RDBMS (Relational Database Management System)** es un sistema de gestión de bases de datos que organiza la información en **tablas relacionadas entre sí**.

### Características principales:

* **Uso de tablas:** los datos se almacenan en filas y columnas.
* **Relaciones entre datos:** permite conectar información mediante llaves primarias y foráneas.
* **Lenguaje SQL:** utiliza SQL para consultar y manipular los datos.
* **Integridad de datos:** asegura que la información sea consistente y válida.

### Ejemplos de RDBMS:

* **PostgreSQL:** usado en aplicaciones robustas y sistemas empresariales.
* **MySQL:** muy común en desarrollo web.
* **SQLite:** utilizado en aplicaciones móviles o sistemas ligeros.

---

## 3. Herramientas y objetos

### Herramientas:

* **DBeaver:** herramienta gráfica para gestionar bases de datos de forma visual.
* **psql:** herramienta de línea de comandos para interactuar con PostgreSQL.

### Objetos de base de datos:

* **Tabla:** estructura principal donde se almacenan los datos en filas y columnas.
* **Vista:** consulta guardada que muestra datos de una o más tablas.
* **Índice:** estructura que mejora la velocidad de búsqueda de datos.
* **Llave primaria:** identifica de manera única cada registro en una tabla.
* **Llave foránea:** permite relacionar una tabla con otra.
